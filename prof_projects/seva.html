Seva Foundation

Stakeholder Fundraising & Strategy Presentation

Overview

Data Cleaning & Preparation Process

A critical part of this project involved transforming raw, inconsistent survey data into a reliable dataset that could support accurate analysis. The data, provided by the Seva Foundation, came from multiple districts and collection periods, which introduced duplicates, inconsistencies, and formatting issues that needed to be addressed before visualization.

Data Processing in Snowflake

I worked with the dataset within Snowflake, where I performed the primary data cleaning using SQL.

Key steps included:

  • Removing duplicate records using DISTINCT and window functions
  • Standardizing formats (e.g., district names, numeric fields)
  • Filtering out incomplete or null entries
  • Structuring the dataset for analysis and visualization

To improve efficiency, I leveraged Snowflake's built-in Copilot to:

  • Generate initial SQL query drafts
  • Refine transformation logic
  • Iterate quickly when debugging or optimizing queries

Data Validation & Refinement in Excel

After cleaning the data in Snowflake, I exported it into Microsoft Excel for additional validation. This also allowed me to preserve the original dataset without making direct changes.

In Excel, I:

  • Used filters to identify anomalies and outliers
  • Applied VLOOKUP to detect duplicate or mismatched records
  • Checked for misspellings and inconsistent naming conventions
  • Reviewed edge cases that required manual inspection

This step helped catch issues that are often easier to identify visually than through SQL alone.

Ensuring Data Reliability

By combining SQL-based cleaning with manual validation, I was able to:

  • Eliminate duplicate and inconsistent records
  • Improve overall data integrity
  • Ensure the dataset was reliable for analysis

Impact on Analysis

This process directly strengthened the final outputs by:

  • Enabling more accurate geographic comparisons
  • Preventing misleading insights caused by data errors
  • Providing a clean foundation for Power BI dashboards
