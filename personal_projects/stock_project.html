Stock Option Cluster Analysis Model

Personal Project

Overview

I used historical stock data from the S&P 500 to glean insights about profit in relation to volume of shares traded. This involved researching stock data, creating measurable metrics, and the various steps to ensure the data was analyzed correctly.

Steps in Analyzing Data

  1. Upload dataset to RStudio from Kaggle
  2. Clean the Data (removing 0s and NAs using R)
  3. Used Distance Matrix to determine ideal number of clusters
  4. Used K-means Clustering, Hierarchical Clustering Analysis, & Dendrograms to analyze data
  5. Drawing conclusions using context of the stock in the S&P 500
